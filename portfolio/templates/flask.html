My TODO Flask Application

The TODO Flask Application is a simple and efficient task management tool designed to help users organize their daily activities. With this application, you can:

  • Add a TODO: Create tasks by entering a title and description, and select a time preference (Day or Night).
  • View Your TODOs: All tasks are displayed in a structured table showing the task number, title, description, time, and available actions.
  • Update or Delete Tasks: Modify or remove tasks with simple action buttons for seamless task management.

Designed with a minimalist approach, the application ensures a user-friendly experience, making it easy for anyone to stay productive.
